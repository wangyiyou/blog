# 1.SecuretCRT8.3 的  FTP操作
---
## (1.)点击file connect sftp session

> 进入sftp 传输模式

### 命令 
```
- 1. put 文件名    ->>> 传送文件到目标机器
- 2. lls ->>>查看当前文件包含那些文件
- 3. lpwd ->>> 查看当前文件所在路径
- 4. cd 文件名->>>到达指定文件
- 5. quit ->>>退出sftp模式

```
